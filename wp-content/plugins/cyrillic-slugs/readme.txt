Automatically transliterates Cyrillic letters in post and page permalinks to their Latin phonetic equivalent.

== Description ==

This plugin will automatically transliterate permalink URLs when you save a new page or post with a title in the Cyrillic script. No configuration needed, just activate it and it works!

== Installation ==

1. Upload `cyr-slugs.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
